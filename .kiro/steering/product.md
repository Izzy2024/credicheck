# Product Overview

## CrediCheck - Credit Reference System

CrediCheck is a SaaS application for quick and secure consultation of negative credit references. The system allows financial analysts to search for credit information by name, ID number, or identification documents.

### Core Features
- **User Authentication**: Secure login system with email/password validation
- **Credit Search**: Search functionality by full name, ID number, or identification documents
- **Record Management**: Add new negative credit references to the database
- **Search History**: Track and review previous searches
- **Dashboard Analytics**: View statistics on daily queries, active references, and match rates

### User Roles
- **Credit Analysts**: Primary users who perform credit reference searches and add new records
- **System Administrators**: Manage user accounts and system configuration

### Language & Localization
- Primary language: Spanish (Colombia)
- UI text, validation messages, and content are in Spanish
- Date formats and currency formatting follow Colombian standards

### Security & Compliance
- SSL secure connections emphasized in UI
- Form validation for data integrity
- Sensitive financial data handling
