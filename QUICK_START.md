# ⚡ Inicio Súper Rápido - CrediCheck

## 🎯 Un Solo Comando

```bash
# Linux/macOS
./dev.sh

# Windows
.\start.ps1
```

¡Eso es todo! 🎉

## 📋 Lo que hace el script automáticamente:

1. ✅ Verifica Node.js y Docker
2. ✅ Levanta PostgreSQL y Redis
3. ✅ Crea archivos de configuración
4. ✅ Instala dependencias
5. ✅ Configura la base de datos
6. ✅ Inicia la aplicación

## 🌐 Acceso a la aplicación:

- **Frontend**: http://localhost:3000
- **API**: http://localhost:3001
- **Base de datos**: http://localhost:5555

## 🛠️ Si algo falla:

```bash
# Ver ayuda
./start.sh help

# Configurar desde cero
./start.sh setup

# Solo base de datos
./start.sh db
```
